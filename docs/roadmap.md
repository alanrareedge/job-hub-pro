# Job Hub Pro Roadmap

## Phase 1: Product Foundation

- Confirm MVP scope
- Define data model
- Set up project structure
- Set up database
- Set up authentication
- Set up multi-tenant business accounts

## Phase 2: Customers and Properties

- Create customer records
- Create property records
- Link customers to properties
- View customer job history
- View property job history

## Phase 3: Jobs

- Create jobs
- Assign jobs to customers and properties
- Add job status
- Add job value
- Add job dates
- Assign operative
- Show job list
- Show job detail page

## Phase 4: Operative Mobile Workflow

- Mobile-friendly job list
- View assigned jobs
- Add job notes
- Upload photos
- Record materials used
- Mark job complete

## Phase 5: Sign-Offs and Payments

- Capture customer sign-off
- Store signature
- Mark job complete
- Record deposit
- Record final payment
- Track outstanding balance

## Phase 6: Audit Trail

- Record key system events
- Show activity history against each job
- Show activity history against each customer

## Phase 7: Pilot

- Test with one trades business
- Watch where they struggle
- Simplify screens
- Remove unnecessary steps
- Improve mobile experience

## Phase 8: Commercial Version

- Go High Level integration
- Stripe payment links
- Repeat work reminders
- Basic customer portal
- Better reporting
- Supplier and materials improvements
